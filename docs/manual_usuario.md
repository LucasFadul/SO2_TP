# Manual de usuario

## Dashboard

El dashboard muestra alarmas con:

- timestamp;
- tipo de alarma;
- IP de origen;
- modulo;
- accion tomada.

## Tipos de alarma principales

- `MODIFICACION_ARCHIVO`
- `USUARIO_SOSPECHOSO`
- `SNIFFER_DETECTADO`
- `FAILED_LOGIN_MULTIPLE`
- `SCANNER_HTTP`
- `MAIL_QUEUE_ALTA`
- `PROCESO_ALTO_CONSUMO`
- `ARCHIVO_TMP_SOSPECHOSO`
- `DDOS_DETECTADO`
- `CRON_SOSPECHOSO`
- `ACCESO_INVALIDO_REPETIDO`
- `CREDENTIAL_STUFFING`
